Overall description: a collection of relatively simple Python scripts with all sort of quant finance use cases ranging from basic financial engineering (e.g. curve construction) to machine learning

#####FinancialMachineLearning_EQ_Feature#####
Inspired by "Advances in Financial Learning" - De Prado. Collects some stock prices from Yahoo Finance, implements data labelling, triple barrier method, features engineering, an XGBOOST model, and some basic back test. Created in 2026

######Order Book#####
A simple script which creates class objects to construct a virtual order book

######Arima-Garch######
Estimates vol using Arch Garch type models

######American Option LSM######
Values american style option using LSM model (Monte Carlo type approach)

#####Curve Bootstrap#####
Zero curve construction

#####Lassor Regression#####
Basic linear regression with Lasso correction

#####Portfolio Allocation######
Optimization techniques for an Equity portoflio, seeking optimal risk return profile

######RNN trading signal######
Generates a trading strategy for EQ stocks using an RNN (LSTM type) model. Uses standard triple barrier method

#####Simple Monte Carlo######
Pricing of european options using MC approach, and test convergence to BS
